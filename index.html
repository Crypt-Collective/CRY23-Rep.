✅ Crypt.Collective website is ready for deployment

Follow these steps to put it online using GitHub Pages:

  1. Download this file as index.html on your computer.
  2. Create a free repository on GitHub.
  3. Upload the index.html file to your repository.
  4. Go to Settings → Pages (on the repo menu).
  5. Under Source, select Deploy from a branch, then select main branch, and click Save.
  6. Your site will appear at:
    https://yourusername.github.io/your-repo-name/
